# Windows Programs To Install

- [Everything](https://www.voidtools.com/) - Instant file search utility
- [7-Zip](https://www.7-zip.org/) - Free file archiver with high compression
- [RivaTuner Statistics Server](https://www.guru3d.com/download/rtss-rivatuner-statistics-server-download/) - Framerate monitoring and OSD overlay
- [ScreenToGif](https://www.screentogif.com/) - Screen recorder that saves as GIF or video
- [StartAllBack](https://www.startallback.com/) - Restores classic Windows 11 taskbar/start menu
- [TreeSize](https://www.jam-software.com/treesize) - Disk space analyzer
