# Mac Programs To Install

- [SoundSource](https://rogueamoeba.com/soundsource/) - Per-app volume control and audio routing
- [Karabiner-Elements](https://karabiner-elements.pqrs.org/) - Keyboard customizer/remapper
- [KeepingYouAwake](https://keepingyouawake.app/) - Prevents Mac from sleeping (Caffeine replacement)
- [Rectangle](https://rectangleapp.com/) - Window manager with keyboard shortcuts and snap areas
